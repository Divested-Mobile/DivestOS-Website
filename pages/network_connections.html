Network Connections¶

Here is a list of expected network connections for security/verification purposes. It should however be assumed incomplete.

NTP¶

  • 2.android.pool.ntp.org - not Google, but a volunteer pool

DNS¶fallback only, cell/Wi-Fi network or user defined 'Private DNS' overrides

  • 9.9.9.9
  • 149.112.112.112
  • 2620:fe::fe
  • 2620:fe::9

Network Connectivity Check¶every connection change, can be disabled in Settings

  • http://connectivitycheck.gstatic.com/generate_204
  • https://www.google.com/generate_204
  • http://www.google.com/gen_204
  • http://play.googleapis.com/generate_204

SUPL¶fallback only, carrier/SIM and emergency calls will override

  • supl.google.com:727X

A-GPS/PSDS¶when acquiring a GPS lock

  • https://xtrapath[1-9].izatcloud.net/{xtra,xtra2,xtra3grc}.bin

Updater¶default weekly, can be disabled in Settings > Updater

  • https://divestos.org/updater.php?base=$BASE&device=$DEVICE&inc=$BUILD_ID
  • https://divestos.org/mirror.php?base=$BASE&file=$FILE
  • https://divestos.org/builds/$BASE/$DEVICE/divested-$VERSION-$DATE-dos-$DEVICE(-$INCREMENTAL).zip

F-Droid Repositories¶default daily, can be disabled in F-Droid > Settings

  • https://f-droid.org/repo/
  • https://divestos.org/fdroid/official/
  • https://divestos.org/fdroid/webview/
  • https://divestos.org/fdroid/unofficial/ - default disabled
  • https://guardianproject.info/fdroid/ - default disabled
  • https://apt.izzysoft.de/fdroid/ - default disabled
  • https://fdroid.bromite.org/fdroid/ - default disabled
