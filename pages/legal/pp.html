Privacy Policy (May 8th, 2018)

User privacy is one of the primary goals of DivestOS. Detailed below is everything you need to know about how your data is handled.

What data we (Spot Communications, Inc.) collect

  • Website
    • What we collect: Page Visited, Referring Page, User Agent, and IP Address
    • How often: On every page visit
    • Why we collect: Used to serve the web pages to users
    • When it will be deleted: When the Apache log files are rotated/deleted
    • What else will we use it for: Nothing else
    • How to anonymize: Visit the site using the Tor Browser
  • Website Analytics
    • What we collect: Page Visited, Stripped Referring Page, Browser Fingerprint (salted with domain), User Agent, IP Address
    • How often: On every page visit
    • Why we collect: Used to determine how popular and roughly how many users we have
    • When it will be deleted: Only the past 200 requests are stored
    • What else will we use it for: Nothing else
    • How to anonymize: Visit the site using the Tor Browser
    • How to disable: The server will not store the request if your browser's 'Do Not Track' option is enabled. This check will be moved to the client in the future.
  • Git Server
    • Coming soon...
  • ROM
    • The ROM does not contain any analytics and any requests are used only for supporting it
  • ROM: Updater
    • What we collect: Device Model, Default Android User Agent, Incremental Build ID, IP Address
    • How often: On every boot and also once per day
    • Why we collect: Used to serve system updates
    • When it will be deleted: When the Apache log files are rotated/deleted
    • What else will we use it for: Will be occasionally used to determine how many users we have and what percent are up-to-date or not
    • How to anonymize: Tor support via Orbot will be added in the future
    • How to disable: Automatic checks can be disabled in Settings > About > DivestOS updates > Preferences > Auto updates check
  • ROM: Captive Portal Check
    • What we collect: Static User Agent, IP Address
    • How often: On every Wi-Fi and cell connection
    • Why we collect: Used to determine if there is an Internet connection
    • When it will be deleted: All requests to gen204.php are never logged
    • What else will we use it for: Nothing else
    • How to disable: $ adb shell settings put global captive_portal_mode 0; #An option to easily toggle this will be added in the future.
  • ROM: DivestOS F-Droid Repos
    • What we collect: Repo Index Requests/App APK Requests/App Icon Requests, F-Droid Version, IP Address
    • How often: On every boot and also once per day
    • Why we collect: Used to serve apps and their updates
    • When it will be deleted: When the Apache log files are rotated/deleted
    • What else will we use it for: Nothing else
    • How to anonymize: Install Orbot and enable 'Use Tor' in F-Droid > Settings
    • How to reduce: Set the 'Automatic update interval' to 'Never' in F-Droid > Settings
    • How to disable: Disable the 'DivestOS' repos in F-Droid > Settings > Repositories
  • App: Hypatia
    • What we collect: Signature Database Requests, IP Address
    • How often: Manually
    • Why we collect: Used to serve signature databases
    • When it will be deleted: When the Apache log files are rotated/deleted
    • What else will we use it for: Nothing else
    • How to anonymize: Install Orbot and enable 'Download over Tor'

What data third parties collect

Third parties are used to support specific features and apps

  • Website Payments
    • Who: Stripe
    • What they receive: Name, Bank Card, E-Mail Address, User Agent, Browser Fingerprint, IP Address
    • How often: When you purchase a ROM from us
    • Why they receive: Used to process the payment
    • How to anonymize: Use a fake name, debit gift card, disposable e-mail address, and connect via a computer at your local library
    • Privacy Policy: Stripe Privacy Policy
  • ROM: Fallback Captive Portal Check
    • Who: Google
    • Description: Used to determine if there is a captive portal, when ours is unavailable
    • What they receive: Static User Agent, IP Address
    • How often: On every Wi-Fi and cell connection
    • How to disable: $ adb shell settings put global captive_portal_mode 0; #An option to easily toggle this will be added in the future.
    • Privacy Policy: Google Privacy Policy
  • ROM: F-Droid Official Repo
    • Who: F-Droid
    • What they receive: Repo Index Requests/App APK Requests/App Icon Requests, F-Droid Version, IP Address
    • How often: On every boot and also once per day
    • Why they receive: Used to serve apps and their updates
    • How to anonymize: Install Orbot and enable 'Use Tor' in F-Droid > Settings
    • How to reduce: Set the 'Automatic update interval' to 'Never' in F-Droid > Settings
    • How to disable: Disable the 'F-Droid' repos in F-Droid > Settings > Repositories
    • Privacy Policy: F-Droid Security Information
