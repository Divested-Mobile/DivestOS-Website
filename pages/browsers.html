Browser Comparison Table

All of the available browsers have each of their own faults as documented in the table below.
While DivestOS includes a Gecko based browser for privacy reasons, Chromium based browsers have many security advantages. It is up to the user to choose their preference.
Fennec based browsers should not be considered for use, as upstream no longer receives security updates. Please use a Fenix based browser instead.
Both lists are sorted by order of recommendation, closer to the top the more we'd recommend.
For more detailed technical information please reference the PrivacyTests website.
Release comparisions are also available for Firefox and Chromium

Firefox calls per-site process isolation Fission and is enabled by default on desktop. Fission does not work on Android. Furthermore Firefox on Android does not take advantage of Android's isolatedProcess flag for completely sandboxing application services.
Firefox calls per-site data isolation first party isolation.
Chromium calls per-site data isolation state partitioning.

WebView Browsers¶

These browsers must be combined with an up to date WebView provider to be secure.
If you are on stock/OEM version of Android or have Google Apps installed, you are likely using the Chrome WebView with extra telemetry.
While WebView browers utilize the Chromium WebView, they cannot offer any per-site process isolation.
