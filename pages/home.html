Key Features

DivestOS is an unofficial soft fork of LineageOS.

Deblobbed

Over 500 proprietary blobs are removed during build time. Removal of these proprietary blobs increase security by reducing attack surface and increase privacy by removing untrusted code.

Increased Security

One of the biggest issues when it comes to aftermarket ROMs is the Linux kernel not being patched against known exploits. We've developed an automated CVE patch checker that is able to apply fixes for over 1,200 vulnerabilities.

Increased Privacy

A custom hosts file created by analyzing over 250 million domains is included that blocks over 200,000 advertising and tracking servers. In addition LineageOS's built in analytics service, CMStats, is also removed.

Free Space Eraser

Extirpater is the first free space eraser for Android. It helps prevent deleted files from being recovered. Perfect for migrating from an unencrypted install or before selling your device.

Malware Scanner

We created Hypatia, the first ever open source realtime malware scanner for Android. Backed by the standard and trusted ClamAV signatures, it can detect over 690k malware signatures.

Privacy Oriented Browser

Mull is a browser based on Mozilla technology that is hardened against advanced fingerprinting techniques thanks to the Tor uplift project and @pyllyukko's user.js project

F-Droid Included

F-Droid is an app store that specializes in delivery of FOSS apps. For the most part all apps available through F-Droid respect your privacy and device.

UnifiedNLP Included

UnifiedNLP is a plugin based Fused location provider. This allows you to enable plugins that can access various offline or online databases to determine your location in a battery saving way without GPS.

Silence Included

Silence, a SMS only fork of Signal, is substituted in place of the default AOSP app. It allows for end-to-end encrypted messages to other Silence users.
