Our Apps

All of the following apps are FOSS and are available via our F-Droid repository. If you are running DivestOS the repository is already added and available to you. For others the repository information is available below. Many are also available in the official F-Droid repository, but may take a few days longer to receive updates.

Hypatia¶A real-time malware scanner (2017)

The first open source real-time malware scanner for Android. Over 7.2 million malware signatures can be detected by default, and configurable up to over 48 million. Real-time scanning can scan average sized files in 5-40ms and has a near zero impact on device performance and battery life. All operations are done locally on the device. Internet is only used to download signature databases. Your files and/or their metadata will never leave your device.

  • Minimum supported Android version: 4.1 / SDK16 / Jelly Bean
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Download: F-Droid, CI Artifacts (debug .apk)
  • Database: Stats & Sources
  • Test File: Have Hypatia running? Save this file to test it!

Extirpater¶A free space eraser (2017)

A high performance free space eraser, the first on Android. Use before selling your device, after enabling encryption, or after deleting many apps/files. Warning! Due to how flash drives work and the partition layout of devices, it'll never be possible to fully fill the drive.

  • Minimum supported Android version: 4.0 / SDK14 / Ice Cream Sandwich
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Download: F-Droid, CI Artifacts (debug .apk)

Mull¶Privacy oriented web browser (2017)

This is a privacy oriented and deblobbed web browser based on Mozilla technology. It enables many features upstreamed by the Tor uplift project using preferences from the arkenfox-user.js project. It is compiled from source and proprietary blobs are removed using scripts by Relan from here.

  • Minimum supported Android version: 5.0 / SDK21 / Lollipop
  • License: MPL-2.0, AGPL-3.0
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Known Issues: workarounds
  • Download: F-Droid
  • Release Comparison: table
  • Recommended Settings: here
  • Independent Comparisions: PrivacyTests

Mulch¶Security oriented web browser (2021)

This is a security oriented and hardened web browser based on Chromium technology. Many anti-features are disabled/removed and a handful security features are enabled/added thanks to patches from the Vanadium project. The source repo also includes prebuilts and makefiles to allow other operating systems to include Mulch as their system WebView.

  • Minimum supported Android version: 8.0 / SDK26 / Oreo
  • License: BSD, GPL-2.0-only with exceptions, Apache-2.0, and Others
  • Source Code: GitHub, GitLab
  • Report Issues: GitHub Issues, GitLab Issues
  • Download: DivestOS F-Droid Repo (must enable repo first!)
  • Release Comparison: table
  • Recommended Settings: here
  • Other ways to use: Mulch System WebView is pre-installed on systems by @MSe1969, @syphyr, @Tomoms, and @steadfasterX and it can also be used on CalyxOS and iodéOS!

GMaps WV¶Unofficial Google Maps WebView Wrapper (2017)

This is a restricted WebView wrapper for accessing the web version of Google Maps. Intended for use when OpenStreetMap isn't enough. Unofficial, not sponsored or endorsed by Google.

  • Minimum supported Android version: 5.0 / SDK21 / Lollipop
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Download: F-Droid, CI Artifacts (debug .apk)

Carrion¶Block spoofed calls (2024)

Automatically rejects all calls failing STIR/SHAKEN verification. Requires official support for Android 11 or higher, working VoLTE, and carrier support.

  • Minimum supported Android version: 11.0 / SDK30 / Red Velvet Cake
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Download: F-Droid, CI Artifacts (debug .apk)

MergedWiFiNLP¶A UnifiedNlp backend (2015)

This is a UnifiedNlp backend for use with databases generated by WiFiDatabaseMerger.

  • Minimum supported Android version: 4.4 / SDK19 / KitKat
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab

MotionLock (alpha)¶Keeps your device locked (2017)

Automatically keeps your device locked when it is face down or hasn't been moved. Only monitors sensors when necessary. Powered by Sensey library licensed under Apache-2.0. Requires device administrator for locking.

  • Minimum supported Android version: 4.0 / SDK14 / Ice Cream Sandwich
  • License: GPL-3.0-or-later
  • Source Code: Codeberg, GitHub, GitLab
  • Report Issues: Codeberg Issues, GitHub Issues, GitLab Issues
  • Download: F-Droid, CI Artifacts (debug .apk)
