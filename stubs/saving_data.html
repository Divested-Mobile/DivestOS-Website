Saving Data

Documented below are various settings to reduce your mobile data usage. Utilizing all of the available settings below and being mindful of your use can allow for moderate usage even on restricted data plans.

System Wide Saver¶

There is a built in 'Data saver' feature. Enabling it restricts the amount of data an app can use while in background and also informs apps that you would like to save data. As of January 2019 only a small handful of apps on F-Droid change how they operate when this is enabled. You can enable it via Quick Settings or Settings > Network > Data usage.

System Wide Cap¶

Data usage can be cutoff completely if a threshold is reached: Settings > Network > Data usage > Billing cycle

AntennaPod¶

By default will only update when on Wi-Fi or manually invoked.

Bitcoin Wallet¶

Will continually use data while running.

Chromium¶

Enabling the 'Data saver' feature routes ALL HTTP (non encrypted) traffic through Google's proprietary compression proxy servers. It also sends the 'Save-Data' HTTP header with all HTTPS connections. Usage of this feature is NOT recommended due to the privacy trade-off.

Conversations¶

  • When connected to a properly maintained XMPP server, data usage should be minimal.
  • Automatic download of attachments is disabled when system 'Data saver' is enabled.
  • Usage can be further reduced via attachment options.

DuckDuckGo Browser¶

An ad/tracker blocker is enabled by default.

E-Mail (AOSP)¶

  • Automatic inbox refresh time interval can be increased/disabled: Settings > [account]
  • Automatic attachment download can be capped/disabled: Settings > [account]

Face Slim¶

  • Images can be disabled.
  • 'Basic' mode can be enabled.

F-Droid¶

By default will only update when on Wi-Fi or manually invoked.

Firefox/Fennec/Mull/Tor Browser¶

  • [3rdParty] Install 'uBlock Origin' add-on to prevent loading of trackers/ads.
  • [3rdParty] Restrict image loading: Settings > Addons > uBlock Origin > Settings > Block media elements larger than
  • [3rdParty] Downloading of fonts can be disabled: Settings > Addons > uBlock Origin > Settings > Block remote fonts
  • Enable Tracking Protection: Settings > Enhanced Tracking Protection > Strict
  • Auto-play of media can be disabled: Settings > Site permissions > Autoplay > Block audio and video

Flym/spaRSS¶

  • Automatic refresh time can be reduced.
  • Automatic refresh can be disabled on mobile.
  • Image precaching is only enabled on Wi-Fi by default.
  • Images can be disabled.

Forecastie¶

Background updates can be disabled.

FOSS Browser¶

  • An ad/tracker blocker is enabled by default.
  • Images can be disabled: Settings > Start control
  • JavaScript can be disabled: Settings > Start control
  • 3rd party requests can be disabled: Settings > Start control
  • Enable 'Request Save Data' via Settings > Start control to send HTTP header informing you would like to save data.

Hypatia¶

Will only update when manually invoked.

Kiwix¶

  • By default will only update when on Wi-Fi or manually invoked.
  • By default warns when clicking external links.
  • Note: Loading 'Download Content' menu takes ~11MB.

Klar/Focus¶

  • An ad/tracker blocker is enabled by default.
  • Blocking of even more trackers can be enabled.
  • Downloading of fonts can be disabled.
  • JavaScript can be disabled.
  • Blocking of images is currently implemented but unavailable.

Lightning¶

  • An ad/tracker blocker can be enabled: Settings > General
  • Images can be disabled: Settings > General
  • JavaScript can be disabled: Settings > General
  • Enable 'Request Save-Data' via Settings > General to send HTTP header informing you would like to save data.

NewPipe¶

  • Resolution can be reduced on mobile, 240P is recommended at ~2MB per minute: Settings > Video & Audio
  • Thumbnail loading can be disabled: Settings > Content

OpenKeychain¶

By default will only update when on Wi-Fi or manually invoked.

Orbot¶

  • Will continually use an insignificant amount of data while running.
  • By default will send fewer padding packets and close circuits earlier to save data.
  • Automatically goes to sleep when not used by any app for a while.

OsmAnd¶

  • Always performs updates checks on app startup.
  • Will only download map/databases when manually invoked.
  • Online maps and images have to be manually enabled.

Privacy Browser¶

  • An ad/tracker blocker is enabled by default.
  • JavaScript is disabled by default.
  • Image loading can be disabled.
  • 3rd party requests can be disabled

RadioDroid¶

  • Favicon downloading can be disabled.
  • Warn on mobile data can be enabled.

RedReader¶

  • Allow thumbnails on Wi-Fi only: Settings > Appearance
  • Precaching can also be disabled: Settings > Cache

Silence¶

Automatic MMS download can be disabled: Settings > Chats and media

Slide¶

Image quality for select hosts can be reduced: Settings > Data saving

Syncthing¶

  • By default will only run when on Wi-Fi.
  • Has an option to respect system 'Data saver': Settings > Run conditions

Twidere¶

  • Automatic refresh time interval can be increased/disabled: Settings > Refresh
  • Media previews can be disabled on mobile: Settings > Network

Wikipedia¶

  • Link preloading can be disabled.
  • Images can be disabled.
  • Stats can be disabled.
  • Note: the data saver/mobile detection seems to be broken.
  • Consider using Kiwix with an offline 'Wikipedia Simple' database (400MB) and only using the Wikipedia app to supplement.
